🏗️ 系统架构

1. 后端 API 层

// pages/api/alliance/products.ts
export default async function handler(req, res) {
  const { platform = 'all', category, minPrice, maxPrice, limit = 20 } = req.query
  
  // 调用淘宝联盟 API
  const taobaoProducts = await callTaoBaoAPI({ ... })
  
  // 调用京东联盟 API  
  const jdProducts = await callJDAPI({ ... })
  
  // 合并并返回数据
  res.json({ products: [...taobaoProducts, ...jdProducts] })
}

🔧 核心功能实现

环境自动识别

useEffect(() => {
  const ua = window.navigator.userAgent.toLowerCase()
  const isWX = ua.indexOf('micromessenger') !== -1
  setIsWechat(isWX)
}, [])

智能转链逻辑

const handlePromotion = async () => {
  if (item.platform === 'taobao') {
    // 淘宝：复制淘口令
    await navigator.clipboard.writeText(item.promoCode)
    alert('淘口令已复制，请打开手机淘宝下单')
  } else if (item.platform === 'jd') {
    if (isWechat) {
      // 微信内：复制链接
      await navigator.clipboard.writeText(item.promoCode)
      alert('京东优惠链接已复制')
    } else {
      // 非微信：直接跳转
      window.open(item.promoCode, '_blank')
    }
  }
}

📋 API 参数说明

GET /api/alliance/products

参数类型默认值说明
platformstring'all'平台筛选：'all' | 'taobao' | 'jd'
categorystring-分类 ID
minPricestring-最低价格
maxPricestring-最高价格
limitnumber20返回数量限制

示例请求

# 获取所有平台商品
curl http://localhost:3000/api/alliance/products

# 只要淘宝商品
curl http://localhost:3000/api/alliance/products?platform=taobao

# 价格 100-500 元的商品
curl http://localhost:3000/api/alliance/products?minPrice=100&maxPrice=500

# 京东商品，最多返回 5 个
curl http://localhost:3000/api/alliance/products?platform=jd&limit=5

🔗 接入真实 API

淘宝联盟（阿里妈妈）

  1. 注册账号 - https://pub.alimama.com/
  2. 创建应用 - 获取 App Key 和 Secret
  3. 调用 API - 使用官方 SDK
const TopClient = require('top-sdk')
const client = new TopClient(appKey, appSecret)

// 获取商品详情
const result = await client.execute('taobao.tbk.item.info.get', {
  num_iid: '123456',
  platform: 1
})

京东联盟（京粉）

  1. 注册账号 - https://union.jd.com/
  2. 申请开发者 - 获取 Access Token
  3. 调用 API - 使用官方 SDK
const JDClient = require('jd-sdk')
const client = new JDClient({
  appKey: 'xxx',
  secret: 'xxx'
})

// 获取推广链接
const link = await client.geneRateLink({
  materialId: '123',
  positionId: '456'
})

⚠️ 注意事项

1. API 调用限制

  • 淘宝联盟：每日调用次数有限制
  • 京东联盟：需要审核通过后才能使用
  • 建议：缓存 API 结果，减少重复调用

2. 佣金更新频率

  • 商品价格会变动
  • 佣金比例可能调整
  • 建议：定期刷新数据（每小时或每天）

3. 用户体验

  • 提供明确的复制提示
  • 支持手动复制
  • 优化移动端体验
